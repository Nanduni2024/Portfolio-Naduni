# Nanduni Sanjana Portfolio

A static portfolio website built with HTML, CSS, and JavaScript. It is ready for free hosting on Vercel with no build step.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Go to https://vercel.com and sign in with GitHub.
3. Click **Add New Project**.
4. Import the portfolio repository.
5. Keep the framework preset as **Other**.
6. Leave the build command empty.
7. Leave the output directory empty.
8. Click **Deploy**.

Vercel will host the site for free and redeploy automatically when you push changes to GitHub.
